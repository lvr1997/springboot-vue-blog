# blog
## 基于SpringBoot+VUE的Blog系统（后端接口部分）
### 功能模块
功能模块|功能介绍|备注
--|--|--
用户登录|登录验证、非登录过滤|
文章管理|文章列表查看、添加删除修改等操作|
留言管理|留言查看、留言审核|
分类管理|对文章分类的管理包括文章标签管理|
。。。|。。。|。。。


---
  
1. 运行mvn.cmd加载项目所需依赖
2. sql表结构在src/main/resource目录下
